本项目主要实现的功能有  
1. 允许点赞（点赞数增加），允许评论（能够在评论区显示）
2. 点击登录/注册，展示相应的登录/注册页面
3. 实现发帖功能（新发的帖子，页面上可以看到）
4. 分页功能，切换不同页，加载不同数据


不足之处:
1.分页功能还有待完善，目前只是以数字展示页数，如果页数过多可能会有显示问题。
2.评论人， 评论图片固定不变
3.未实现取消赞， 解决办法 可在数据里保存赞状态
4.模态框弹出时，界面会抖动
等等
